# Terraform GCP Template Documentation

Project-owned documentation for `terraform-gcp-template` stays directly under `docs/`.
Foundation-owned documentation inherited from `ai-dev-foundation` stays under
[`docs/foundation/`](foundation/).

| Path | Ownership | Purpose |
|------|-----------|---------|
| [foundation/](foundation/) | inherited from `ai-dev-foundation` | reusable foundation guidance, templates, and foundation ADRs |
| [adr/](adr/) | `terraform-gcp-template` | Terraform-template decisions and local ADR index |
| [handoff.md](handoff.md) | `terraform-gcp-template` | mutable restart point for template inheritance work |

Use [`docs/foundation/guides/project-documentation.md`](foundation/guides/project-documentation.md)
for the reusable documentation structure and update-trigger guidance.
